# 1-06-01（GBM10601）

# 企业负责人

在企业中, 担任领导职务并具有决策、管理职权的人员。

本小类包括下列职业:

企业董事

1-06-01-02企业经理

国有企业中国共产党组织负责人

# 1-06-01-01 企业董事

在企业中, 具有最高决策权的董事会组成人员。

主要工作任务:

负责召集股东大会, 向股东大会报告工作, 执行股东大会决议;  
决定企业经营计划和投资方案;  
审定企业年度财务预决算方案、 利润分配方案和弥补亏损方案, 增加或减少注册资本方案及其规定职权范围内的其他方案;  
制定企业合并、 分立、 解散等职权规定的方案;  
聘任或解聘企业经理, 并根据经理

提名, 聘任或解聘企业副经理、 财务负责人, 决定其报酬事项;

审议企业内基本制度。

# 1-06-01-02 企业经理

在企业中, 经董事会或出资人聘任, 或经职代会选举, 或经上级任命的企业负责人。

主要工作任务:

执行董事会和个人独资、 合伙制企业出资人决议或职工代表大会作出的决定;  
领导企业日常经营管理工作;  
拟定或决定企业内部机构设置;  
按公司规定的职权范围, 行使人事聘任、 解聘或任免权;  
决定或议定企业的基本规章制度;  
行使相关法律规定的其他职权。

# 1-06-01-03 国有企业中国共产党组织负责人

在国有企业中国共产党组织中, 担任领导职务的人员。

主要工作任务:

负责召集并主持企业党组织会议,组织传达、 贯彻落实党和国家的路线、 方针、 政策以及上级党委的决议, 讨论安排党

组织的工作计划并组织实施, 依规讨论决定企业重大事项;

负责企业党组织的思想、 组织、 制度、 作风和能力建设以及企业领导班子的思想政治建设, 检查企业党组织决议的执行情况, 定期向党的委员会和党员大会及上级党委报告工作;  
按照法定程序考察、 推荐公司中层以上管理人员, 支持股东会、 董事会、 监事会和经营管理者依法行使职权;  
领导企业职工代表大会和工会、 共青团等群众组织。
