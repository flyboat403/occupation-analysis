# 47-4091.00 - Segmental Pavers

**Quick Facts:**
- SOC Code: 47-4091.00
- Also called: Cutter, Paver, Paver Installer, Paver Layer, Paving Stone Installer, Segmental Wall Installer

## Summary

Lay out, cut, and place segmental paving units. Includes installers of bedding and restraining materials for the paving units.

|------------|----------|------|

| 84 | Core | Supply and place base materials, edge restraints, bedding sand and jointing sand. |
| 82 | Core | Discuss the design with the client. |
| 80 | Core | Set pavers, aligning and spacing them correctly. |
| 78 | Core | Sweep sand into the joints and compact pavement until the joints are full. |

| 72 | Core | Sweep sand from the surface prior to opening to traffic. |
| 56 | Core | Resurface an outside area with cobblestones, terracotta tiles, concrete or other materials. |
