# 51-4034.00 - Lathe and Turning Machine Tool Setters, Operators, and Tenders, Metal and Plastic

**Quick Facts:**
- SOC Code: 51-4034.00
- Also called: CNC Lathe Operator (Computer Numerical Control Lathe Operator), CNC Setup Operator (Computer Numerical Control Setup Operator), Lathe Operator, Machine Operator, Numerical Control Operator (NC Operator), Screw Machine Operator, Screw Machine Tool Setter, Setup Operator, Turn Operator

## Summary

Set up, operate, or tend lathe and turning machines to turn, bore, thread, form, or face metal or plastic materials, such as wire, rod, or bar stock.
