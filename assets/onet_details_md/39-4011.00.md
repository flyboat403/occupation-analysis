# 39-4011.00 - Embalmers

**Quick Facts:**
- SOC Code: 39-4011.00
- Also called: Embalmer, Licensed Embalmer, Trade Embalmer

## Summary

Prepare bodies for interment in conformity with legal requirements.

## Tasks

| Importance | Category | Task |
|------------|----------|------|
| 96 | Core | Conform to laws of health and sanitation and ensure that legal requirements concerning embalming are met. |
| 94 | Core | Apply cosmetics to impart lifelike appearance to the deceased. |
| 94 | Core | Join lips, using needles and thread or wire. |
| 92 | Core | Close incisions, using needles and sutures. |

| 92 | Core | Clean and disinfect areas in which bodies are prepared and embalmed. |
| 91 | Core | Dress bodies and place them in caskets. |
| 91 | Core | Make incisions in arms or thighs and drain blood from circulatory system and replace it with embalming fluid, using pump. |
| 90 | Core | Remove the deceased from place of death and transport to funeral home. |
| 89 | Core | Perform the duties of funeral directors, including coordinating funeral activities. |
| 89 | Core | Attach trocar to pump-tube, start pump, and repeat probing to force embalming fluid into organs. |

| 88 | Core | Pack body orifices with cotton saturated with embalming fluid to prevent escape of gases or waste matter. |
