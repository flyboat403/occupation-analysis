# 15-1299.01 - Web Administrators

**Quick Facts:**
- SOC Code: 15-1299.01
- Also called: Corporate Webmaster, Web Administrator, Web Content Coordinator, Web Content Manager, Web Director, Web Manager, Web Site Manager, Web Technologies Administrator, Webmaster

## Summary

Manage web environment design, deployment, development and maintenance activities. Perform testing and quality assurance of web sites and web applications.
