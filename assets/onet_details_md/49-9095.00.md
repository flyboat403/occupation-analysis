# 49-9095.00 - Manufactured Building and Mobile Home Installers

**Quick Facts:**
- SOC Code: 49-9095.00
- Also called: Delivery Crew Worker, Mobile Home Installer, Mobile Home Laborer, Mobile Home Set-Up Person, Modular Set Crew Member, Set Up Technician

## Summary

Move or install mobile homes or prefabricated buildings.

## Tasks

|------------|----------|------|
