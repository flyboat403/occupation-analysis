# 49-9064.00 - Watch and Clock Repairers

**Quick Facts:**
- SOC Code: 49-9064.00
- Also called: Antique Clock Repairer, Clock Repair Technician, Clock Repairer, Watch and Clock Repairer, Watch Estimator, Watch Repair Person, Watch Repair Technician, Watch Repairer, Watch Technician (Watch Tech)

## Summary

Repair, clean, and adjust mechanisms of timing instruments, such as watches and clocks. Includes watchmakers, watch technicians, and mechanical timepiece repairers.

## Tasks

| Importance | Category | Task |
|------------|----------|------|
| 92 | Core | Clean, rinse, and dry timepiece parts, using solutions and ultrasonic or mechanical watch-cleaning machines. |
| 91 | Core | Adjust timing regulators, using truing calipers, watch-rate recorders, and tweezers. |
| 91 | Core | Reassemble timepieces, replacing glass faces and batteries, before returning them to customers. |
| 90 | Core | Disassemble timepieces and inspect them for defective, worn, misaligned, or rusty parts, using loupes. |
| 89 | Core | Oil moving parts of timepieces. |
| 89 | Core | Estimate repair costs and timepiece values. |
| 89 | Core | Repair or replace broken, damaged, or worn parts on timepieces, using lathes, drill presses, and hand tools. |
| 84 | Core | Test timepiece accuracy and performance, using meters and other electronic instruments. |
| 83 | Core | Perform regular adjustment and maintenance on timepieces, watch cases, and watch bands. |
| 82 | Core | Order supplies, including replacement parts, for timing instruments. |
| 81 | Core | Gather information from customers about a timepiece's problems and its service history. |
| 79 | Core | Test and replace batteries and other electronic components. |
| 74 | Core | Record quantities and types of timepieces repaired, serial and model numbers of items, work performed, and charges for repairs. |
| 71 | Core | Demagnetize mechanisms, using demagnetizing machines. |
| 69 | Core | Fabricate parts for watches and clocks, using small lathes and other machines. |
