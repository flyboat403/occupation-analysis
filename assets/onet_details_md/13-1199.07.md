# 13-1199.07 - Security Management Specialists

**Quick Facts:**
- SOC Code: 13-1199.07
- Also called: Physical Security Engineer, Physical Security Specialist, Security Advisor, Security Analyst, Security and Workplace Violence Consultant, Security Consultant, Security Management Consultant, Security Specialist

## Summary

Conduct security assessments for organizations, and design security systems and processes. May specialize in areas such as physical security or the safety of employees and facilities.
