# 33-1091.00 - First-Line Supervisors of Security Workers

**Quick Facts:**
- SOC Code: 33-1091.00
- Also called: Campus Safety Chief, Public Safety Manager, Public Safety Supervisor, Security Chief, Security Director, Security Guard Supervisor, Security Lieutenant, Security Shift Supervisor, Security Supervisor, Shift Supervisor

## Summary
