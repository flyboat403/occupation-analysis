# 15-1241.00 - Computer Network Architects

**Quick Facts:**
- SOC Code: 15-1241.00
- Also called: Computer Network Technologist, Network Analyst, Network Consultant, Network Systems Consultant, Networking Systems Engineer, Solutions Architect, System Programmer, Systems Engineer, WAN Engineer (Wide Area Network Engineer), Wireless Network Engineer

## Summary

Design and implement computer and information networks, such as local area networks (LAN), wide area networks (WAN), intranets, extranets, and other data communications networks. Perform network modeling, analysis, and planning, including analysis of capacity needs for network infrastructures. May also design network and computer security measures. May research and recommend network and data communications hardware and software.
