# 51-4035.00 - Milling and Planing Machine Setters, Operators, and Tenders, Metal and Plastic

**Quick Facts:**
- SOC Code: 51-4035.00
- Also called: CNC Licensed Mill Operator (Computer Numerical Control Licensed Mill Operator), CNC Mill Operator (Computer Numerical Control Mill Operator), CNC Mill Operator (Computerized Numerical Control Mill Operator), CNC Mill Set Up Operator (Computerized Numerical Control Mill Set Up Operator), Machine Operator, Machine Set Up Operator, Mill Operator, Miller, Milling Operator, Setup Man

## Summary

Set up, operate, or tend milling or planing machines to mill, plane, shape, groove, or profile metal or plastic work pieces.

## Tasks
