# 51-6063.00 - Textile Knitting and Weaving Machine Setters, Operators, and Tenders

**Quick Facts:**
- SOC Code: 51-6063.00
- Also called: Knitter, Knitting Machine Operator, Loom Fixer, Machine Operator, Operator, Tufting Machine Operator, Tufting Operator, Warp Knit Operator, Weaver, Winder Operator

## Summary

Set up, operate, or tend machines that knit, loop, weave, or draw in textiles.
