# 76 新闻传播大类

# 7601 新闻出版类

专业代码 760101

专业名称 出版商务

基本修业年限 三年

# 职业面向

面向出版物发行员、营销员、商品营业员、电子商务师等职业，阅读顾问、出版物销售、收银、仓储管理、网店运营等岗位（群）。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和出版发行行业的出版物发行、市场营销、电子商务等知识，具备出版物市场营销、出版物电子商务、新媒体营销等能力，具有工匠精神和信息素养，能够从事出版物线下线上营销和文化产品客户服务等工作的技术技能人才。

# 主要专业能力要求

1. 具有传统出版物发行进销存管理、图书分类与陈列的能力；  
2. 具有识别正规出版物与非法出版物的能力；  
3. 具有出版物导购、销售、收银、订单管理的能力；  
4. 具有网上书店新品上架、信息编辑、商品更新和售后服务的能力；  
5. 具有初步市场调研和出版物网络营销的能力；  
6. 具有初步出版商务沟通和客户服务的能力；  
7. 具有将大数据、人工智能等现代信息技术应用于出版领域的能力；  
8. 具有主流电商和新媒体渠道实际操作的能力；  
9. 具有终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：出版物识别、图书分类学、出版物市场调查与分析、出版物市场营销。

专业核心课程：出版物流管理实务、出版物宣传与广告制作、出版物发行业务洽谈、

出版物销售实务、出版电子商务、网络编辑实务、新媒体营销、短视频运营。

实习实训：对接真实职业场景或工作情境，在校内外进行仓储管理、模拟书店运营、出版电子商务、数字出版物制作等实训。在各类书店、图书发行企业、出版物卖场等企业或场所进行岗位实习。

# 职业类证书举例

职业技能等级证书：网店运营推广、新媒体运营、数字媒体交互设计

# 接续专业举例

接续高职专科专业：出版商务、出版策划与编辑

接续高职本科专业：市场营销、网络与新媒体

接续普通本科专业：市场营销、网络与新媒体
