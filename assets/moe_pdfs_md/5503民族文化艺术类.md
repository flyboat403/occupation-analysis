# 5503 民族文化艺术类

专业代码 550301

专业名称 民族表演艺术

基本修业年限 三年

# 职业面向

面向民族民间艺术传承人员等职业，民族乐器演奏、歌唱、舞蹈、群众文化活动服务、文化艺术培训等岗位（群）。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和中华优秀传统文化、民族音乐、民族舞蹈、民族艺术基本理论、民族艺术表演等知识，具备文艺创作与表演、群众文体活动组织、文化艺术培训等能力，具有工匠精神和信息素养，能够从事民族乐器演奏、民族舞蹈表演、音乐表演、歌唱表演、文化活动组织与策划、民族民间艺术传承等工作的高素质技术技能人才。

# 主要专业能力要求

1. 具有表演民族传统音乐、声乐、器乐、舞蹈、表演的能力；  
2. 具有运用声乐、器乐、舞蹈表演技巧，塑造民族艺术等形象的能力；  
3. 具有艺术鉴赏和艺术实践的能力；  
4. 具有组织和策划合唱、合奏、舞蹈演出活动的能力；  
5. 具有传承、展示、保护、创新艺术的表演和实施指导的能力；  
6. 具有一定的信息技术和多媒体技术运用的能力；  
7. 具有较好的语言表达、沟通协调和团队合作的能力；  
8. 具有探究学习、终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：乐理、视唱练耳、和声、音乐简史与鉴赏、民族音乐基础理论、地区民族艺术赏析、表演基础。

专业核心课程：民族乐器演（合）奏、少数民族舞蹈与创编、民族技艺传习、声乐演唱、钢琴演奏、舞台表演、合唱与指挥、音乐教学法。

实习实训：对接真实职业场景或工作情境，在校内外进行舞蹈表演、声乐演唱技能、器乐演奏技能、舞台表演、音乐教学法等实训。在演艺企业、文艺院团、文化馆（站）、

艺术培训机构等单位进行岗位实习。

# 职业类证书举例

职业资格证书：教师资格、演出经纪人员资格

职业技能等级证书：器乐艺术指导

# 接续专业举例

接续高职本科专业举例：音乐表演、舞蹈表演与编导

接续普通本科专业举例：音乐表演、舞蹈表演

4. 具有同时使用国家通用语言文字和少数民族语言文字开展教育教学的能力；  
5. 具有使用国家通用语言文字和少数民族语言文字进行文学创作和新闻报道的能力；  
6. 具有对新知识、新技能等数字化技能的学习能力和创新创业的能力；
