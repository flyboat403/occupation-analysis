# 6802 包装类

专业代码 680201

专业名称 包装设计与制作

基本修业年限 三年

# 职业面向

面向包装设计制作、包装生产、包装制品质量控制等岗位（群）。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和包装基础、包装设计、三大构成、机械基础等知识，具备包装产品设计、包装产品制作、包装生产设备操作和包装制品质量检测等能力，具有工匠精神和信息素养，能够从事包装产品设计制作、生产设备操作、产品质量控制和产品质量检测等工作的技术技能人才。

# 主要专业能力要求

1. 具有包装结构设计制作的基本能力；  
2. 具有包装装潢设计制作的基本能力；  
3. 具有包装生产设备操作的基本能力；  
4. 具有包装制品质量检测和控制的能力；  
5. 具有协助开展环保和安全生产监控的能力；  
6. 掌握包装、印刷领域的法律法规，具有依法从事工作的能力；  
7. 具有初步将物联网、大数据等现代信息技术应用于包装设计与制作领域的能力；  
8. 具有终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：包装基础、色彩构成、平面构成、立体构成、机械基础等。

专业核心课程：包装材料、图形图像处理、包装工艺、包装印刷、包装计算机辅助设计、包装质量控制与检测等。

实习实训：对接真实职业场景或工作情境，在校内外进行包装设计、包装印刷、图形图像处理、计算机辅助设计等实训。在校外包装生产企业进行包装生产、包装质量检测、包装质量控制等岗位实习。
