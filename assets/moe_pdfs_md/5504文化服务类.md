专业名称 文化创意与策划

基本修业年限 三年

# 职业面向

面向数字文化创意设计、文化活动服务与指导、工艺美术与创意设计、数字文化创意内容制作服务、新型媒体服务等岗位（群）。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和创意策划、新媒体运营、基础设计与视频制作等知识，具备创意统筹、运营推广、文创产品开发等能力，具有工匠精神和信息素养，能够从事文化活动策划与组织、新媒体运营、文创产品设计与创意策划、短视频创意指导与创作等工作的高素质技术技能人才。

# 主要专业能力要求

1. 具有文化创意活动中的创新能力与实践能力；  
2. 具有文化活动策划、品牌推广、文创产品推介等所需的策划与组织合作能力；  
3. 具有文创产品开发中所需的创意统筹与产品架构能力；  
4. 具有文创产品设计制图中所需的数字技术能力与设计表现能力；  
5. 具有新媒体运营所需的视觉艺术素养和宣传、运营能力；  
6. 具有视频创意制作所需的拍摄技巧、脚本设计、视频剪辑能力；  
7. 具有探究学习、终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：创意学基础、文化产业概论、文创策划、展览展示策划、文创企业管理实务、设计软件应用。

专业核心课程：创意文案写作实务、图形语言、网络传播与策划、短视频创意制作、文创产品设计、文化活动策划实务。

实习实训：对接真实职业场景或工作情境，在校内外进行文化活动策划与推广、新媒体运营策划、文创产品策划与开发等实训。在文化艺术、新媒体、创意设计行业的企事业单位、文化创意相关行业部门进行岗位实习。

专业核心课程：档案管理实务、文献编目与信息描述、信息服务与用户、文献修复与装裱技术、档案文献编研实务、文献信息检索与利用技术、数字图书馆技术与方法、数字档案馆技术与方法。

实习实训：对接真实职业场景或工作情境，在校内外进行档案整理、文献编目、缩微复制、图书馆活动组织策划、图书馆用户调研、数字化加工项目等实训。在图书馆（室）、档案馆（室）、图书公司和档案服务公司等单位进行岗位实习。

专业名称 石窟寺保护技术

基本修业年限 三年

# 职业面向

面向不可移动文物保护专业人员、文物修复师等职业，石窟寺及摩崖石刻、壁画彩塑文物保护等岗位（群）。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和石窟寺文物保护修复专业知识，具备开展文物调查、信息采集、保护修复等能力，具有工匠精神和信息素养，

能够从事石窟寺及摩崖石刻、壁画彩塑等文物的勘察、监测、保护、修复、数字化采集及日常保养等工作的高素质技术技能人才。

# 主要专业能力要求

1. 具有对石窟寺及摩崖石刻文物进行现状调查，识别文物病害，使用相关软件绘制规范的文物病害图，编写文物病害档案的能力；  
2. 具有开展石窟寺文物制作工艺调查、病害监测、环境调查的能力；  
3. 具有对石窟寺文物进行现场勘察、采集记录文物信息及取样、制样、检测分析等能力，能承担实验室与现场试验基本工作；  
4. 具有对石窟寺文物进行岩体加固、洞窟渗漏水治理、清洗、壁画回贴加固、灌浆、粘接、补全等保护修复工作，以及操作常用设备和开发、制作特殊修复工具的能力；  
5. 具有编写文物保护修复档案及辅助编写文物保护修复方案的能力；  
6. 具有熟练使用文字、表格、图像处理等软件，运用文物考古、保护工程摄影摄像方法及数字化处理技术的能力；  
7. 具有一定的石窟寺文物保护修复项目管理能力；  
8. 具有探究学习、终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：中国古代建筑史、考古学概论、中国石窟寺考古与艺术、中国古代石刻概论、文物学概论、地质学基础、文物保护修复基础及材料、文物政策法规及技术标准、文物保护环境概论。

专业核心课程：文物绘图、古代壁画制作技艺、古代雕塑制作技艺、石窟寺调查与监测技术、石窟寺保护工程技术、石质文物保护修复技术、壁画彩塑保护修复技术、文物数字化技术。

实习实训：对接真实职业场景或工作情境，在校内外进行石窟寺文物病害调查、保护修复、数字化技术运用等实训。在石窟寺保护单位、考古机构、博物馆、古建筑保护机构等单位进行岗位实习。
