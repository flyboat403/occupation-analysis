# 6803 印刷类

专业代码 680301

专业名称 印刷媒体技术

基本修业年限 三年

# 职业面向

面向印前处理和制作员、印刷操作员、印后制作员等职业。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和印刷色彩、印刷材料、印刷工艺等知识，具备印前图文处理及输出、印刷与印后设备操作、印刷质量检测与控制等能力，具有工匠精神和信息素养，能够从事印刷品的设计制作、加工服务、客户服务等工作的技术技能人才。

# 主要专业能力要求

1. 具有对印刷品进行色彩测量、调整、控制和专色墨调配的能力；  
2. 具有常用印刷材料识别、检测与选用的能力；  
3. 具有印前图文处理，印刷品完稿设计、拼版和输出的能力；  
4. 具有印刷及印后设备操作运行、故障排除及维护保养的能力；  
5. 具有数字印刷流程设计、数字制版、数字印刷和数字印后操作的能力；  
6. 具有初步的印刷质量检测与控制的能力；  
7. 具有将大数据、人工智能等现代信息技术应用于印刷领域的能力；  
8. 具有终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：印刷概论、印刷色彩基础、印刷材料识别与选用、印刷工艺等。

专业核心课程：图文信息处理、色彩管理、计算机直接制版与数字打样、印刷机结构与调节、传统印刷设备操作、印后加工设备操作、数字印刷设备操作、印刷数字工作流程、印刷质量检测与控制等。

实习实训：对接真实职业场景或工作情境，在校内外进行印前图文处理、印前制作和输出、传统印刷、数字印刷、印后加工等实训。在出版单位、广告公司、印刷包装企业等单位进行岗位实习。
