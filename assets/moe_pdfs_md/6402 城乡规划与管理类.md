# 6402 城乡规划与管理类

专业代码 640201

专业名称 城镇建设

基本修业年限 三年

# 职业面向

面向村镇建设规划管理、村镇建筑与市政工程施工等技术领域。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和土木工程力学、建筑构造、建筑材料、工程测量、建筑与市政工程施工、城镇建设规划管理等知识，具备村镇建设规划实施、建筑与市政工程施工操作等能力，具有工匠精神和信息素养，能够从事工程测量与地籍测绘、村镇建设规划管理、城镇建筑与市政工程施工操作和施工作业管理等工作的技术技能人才。

# 主要专业能力要求

1. 具有村镇规划设计文件阅读和建筑工程施工图识读能力；  
2. 具有使用测量仪器进行工程测量与地籍测绘的能力；  
3. 具有识别常见建筑材料的能力；  
4. 具有简单建筑工程 CAD 绘图、BIM 建模的能力；  
5. 具有村镇建筑与市政工程施工操作和施工作业管理的能力，以及城镇建设管理的基本能力；  
6. 具有协助编制工程招投标文件、协助组织实施建筑工程招投标活动的能力；  
7. 具有在村镇建设过程中综合运用绿色生产、环境保护、质量管理和法律法规等知识的能力；  
8. 具有专业信息技术能力和初步数字化技能；  
9. 具有终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：建筑材料、建筑 CAD、建筑构造与识图、建筑力学与结构。

专业核心课程：工程测量与地籍测绘、城镇规划、BIM 建模、建筑与市政工程施工、工程质量与安全管理、村镇建设管理与建设法规、市政工程规划与建设、招投标与合同管理。

实习实训：对接真实职业场景或工作情境，在校内外进行城镇规划实施、工程测量、建筑工程工种工艺操作等实训。在城镇规划设计、城镇规划管理职能部门和建筑施工企业等单位进行岗位实习。

# 职业类证书举例

职业技能等级证书：建筑工程施工工艺实施与管理、不动产数据采集与建库、建筑信息模型（BIM）

# 接续专业举例

接续高职专科专业举例：村镇建设与管理、建筑工程技术、建设工程管理

接续高职本科专业举例：城乡规划、建筑工程、建设工程管理

接续普通本科专业举例：城乡规划、土木工程
