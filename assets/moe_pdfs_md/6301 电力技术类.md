# 6301 电力技术类

专业代码 630101

专业名称 发电厂及变电站运行与维护

基本修业年限 三年

# 职业面向

面向发电集控值班员、变配电运行值班员、继电保护员等职业，发电厂电气运行、变电运行、继电保护运维等岗位（群）。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和电工、电子、电气测量、电机运行及控制、电气二次回路等知识，具备发电厂及变电站设备巡视、倒闸操作、电气试验、继电保护运维等能力，具有工匠精神和信息素养，能够从事发电厂及变电站电气运行、电气设备倒闸操作、常规电气试验、继电保护运维等工作的技术技能人才。

# 主要专业能力要求

1. 具有电工、电子基本电路安装及调试的能力；  
2. 具有发电厂及变电站一、二次回路识图的能力；  
3. 具有电机与变压器常用试验、电机常用控制回路安装及调试的能力；  
4. 具有发电厂、变电站设备巡视及常见故障处理、电气设备倒闸操作的能力；  
5. 具有发电厂、变电站常规电气试验的能力；  
6. 具有继电保护与自动装置安装、配置、运行与维护的能力；  
7. 具有适应发电厂、变电站设备运行与维护行业数字化发展需求的基本数字技能；  
8. 具有绿色生产、环境保护、安全防护、质量管理的能力；  
9. 具有终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：电工技术基础与技能、电子技术基础与技能、工程制图与 CAD、电力安全技术。

专业核心课程：电气测量技术及应用、电机运行及控制、电力系统基础、发电厂及

变电站电气设备运维、电力系统继电保护与自动装置运维、电气试验、电气运行技术及应用、电气二次回路识图与运维。

实习实训：对接真实职业场景或工作情境，在校内外进行电机运行及控制实训、电气设备安装与调试实训、电气运行仿真实训、电气试验、继电保护装置调试实训等综合实训。在电力、热力生产和供应行业的火力发电厂、新能源发电厂（站）、抽水蓄能电站、智能变电站、热电厂等单位进行岗位实习。

# 职业类证书举例

职业技能等级证书：变配电运维、变电一次安装、变电二次安装

# 接续专业举例

接续高职专科专业举例：发电厂及电力系统、电力系统继电保护技术、供用电技术、光伏工程技术、风力发电工程技术

接续高职本科专业举例：电力工程及自动化、智能电网工程技术、新能源发电工程技术

接续普通本科专业举例：电气工程及其自动化
