# 职业面向

面向计量员、质检员、试验员等职业，计量器具的计量检定与维保、工业产品检验检查、工业产品工艺性能实验、工业产品质量分析等岗位（群）。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和计量测试基础理论、计量器具使用及维保等知识，具备计量、质检、试验基础数据分析和计算，计量测试器具设备规范使用等能力，具有工匠精神和信息素养，能够从事计量器具的计量检定、检查、修理、测试，使用设备、工具对工业产品进行检验、检查，以及使用设备、仪器进行工艺性试验或产品性能试验等工作的技术技能人才。

# 主要专业能力要求

1. 具有计量器具及设备规范操作的能力；  
2. 具有计量、质检、试验基础数据分析和计算的能力；  
3. 具有使用设备、工具或运用感官检验、检查产品质量，以及使用设备、仪器进行工艺性试验或产品性能试验的能力；  
4. 具有协助完成产品的质量检测、计量测试、标准认证等工作的能力；  
5. 具有适应产业数字化发展需求的基本数字技能；

7. 具有终身学习和可持续发展的能力。
