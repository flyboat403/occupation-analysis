# 79 公共管理与服务大类

# 7901 公共事业类

专业代码 790101

专业名称 社会工作事务

基本修业年限 三年

# 职业面向

面向社会工作者和社区工作者等职业，社会工作助理等岗位（群）。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和社会调查、活动策划与组织、个案工作、小组工作、社区工作等知识，具备收集和统计服务数据、贯彻宣传政策法规、联络志愿者等能力，具有公益精神和信息素养，能够从事接待来访服务对象、服务档案管理、服务场地管理、常规服务执行、志愿者对接等工作的技术技能人才。

# 主要专业能力要求

1. 具有社会工作价值观及专业伦理实践的能力；  
2. 具有开展活动宣传、节目彩排、会场布置、物品筹备、邀请嘉宾等系列准备工作的能力；  
3. 具有社会工作各类应用文书写作的能力；  
4. 具有整理相关数据和资料，分析服务对象基本问题及需求的能力；  
5. 具有贯彻执行国家相关政策法规，认真落实相关工作部署，并开展相关宣传工作的能力；  
6. 具有收集、分析基层社区基本问题及需求的能力；  
7. 具有运用数字化技能处理专业基础信息的能力；  
8. 具有终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：社会工作基础、社会工作政策法规与标准、社会工作活动策划与组织、应用文写作。

专业核心课程：社会调查方法、个案工作案例与分析、小组工作案例与分析、社区

工作案例与分析、社区服务与社区治理、社会工作实务。

实习实训：对接真实职业场景或工作情境，在校内外进行个案工作、小组工作、社区工作等实训。在校外社会工作服务组织、街道（乡镇）、社区（村）、社会服务类企事业单位等单位进行岗位实习。

# 职业类证书举例

职业技能等级证书：社区治理、失智老年人照护、社会心理服务、生涯规划指导、家务管理

# 接续专业举例

接续高职专科专业举例：社会工作、民政服务与管理、社区管理与服务

接续高职本科专业举例：社会工作、民政管理、智慧社区管理

接续普通本科专业举例：社会工作

专业代码 790102

专业名称 社区公共事务管理

基本修业年限 三年

# 职业面向

面向社区工作者职业，社会政策宣传与落实、公共事务受理与核查、社区服务活动筹备与开展等岗位（群）。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和社区法规政策、政务服务、社区福利服务、社区志愿服务等知识，具备宣传动员、政务受理、信息采集、组织协调等能力，具有工匠精神和信息素养，能够从事政策宣传与落实、公共事务受理与核查、社区服务活动筹备与开展等工作的技术技能人才。

# 主要专业能力要求

1. 具有贯彻执行、宣传落实法规政策的能力；  
2. 具有协助人民政府或者派出机关受理与核查公共事务的能力；  
3. 具有筹备与开展社区便民服务和志愿服务的能力；  
4. 具有开展居（村）民自治活动的能力；

5. 具有使用智能设备采集图像视频和上报基础信息与社情民意的能力；  
6. 具有化解和排查矛盾纠纷与安全隐患的能力；  
7. 具有终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：社区工作认知、公文写作基础、社区法规与政策、新媒体技术与应用。

专业核心课程：社区宣传实务、社区沟通与调解、公共事务办理、社区活动组织与实施、信息采集与处理、社区志愿服务、社区福利服务、社区自治实务。

实习实训：对接真实职业场景或工作情境，在校内外进行信息采集与录入、社区管理系统操作、社区事务办理、社区便民服务、社区志愿服务等实训。在村（社区）综合服务中心、公共服务中心、社区社会组织等单位进行岗位实习。

# 职业类证书举例

职业技能等级证书：社区治理

# 接续专业举例

接续高职专科专业举例：社区管理与服务

接续高职本科专业举例：智慧社区管理

专业代码 790103

专业名称 社会福利事业管理

基本修业年限 三年

# 职业面向

面向孤残儿童护理员、失智老年人照护员等职业。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和社会福利服务基础知识与特殊困难社会群体康复服务、福利服务、需求评估等知识，具备社会福利政策法规运用与特殊困难社会群体康复服务、福利服务、需求评估、项目申报等能力，具有人文精神和信息素养，能够从事基层社会福利事业领域的孤残儿童护理、失智老年人照护服务等工作的技术技能人才。

# 主要专业能力要求

1. 掌握一定的社会福利服务专业知识与社会福利政策法规，具有社会福利政策法规运用的能力；  
2. 掌握孤残儿童、失智老年人福利服务的相关服务理念和知识技能，具有开展社会福利服务的能力；  
3. 掌握孤残儿童、失智老年人康复辅具的基本类型、使用方法与技巧，具有日常照护、保健与康复服务的能力；  
4. 具有评估孤残儿童、失智老年人社会福利服务需求的能力；  
5. 掌握社会福利机构日常运营的工作流程，具有日常运营与管理福利机构的能力；  
6. 掌握社会福利服务项目的申报过程，具有孤残儿童、失智老年人等社会福利项目申报的能力；  
7. 具有终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：社会福利政策与法规、特殊困难群体心理发展、特殊困难人群能力需求评估。

专业核心课程：孤残儿童和失智老年人人际沟通、孤残儿童和失智老年人生活照护、孤残儿童和失智老年人膳食与营养、孤残儿童和失智老年人保健与康复、孤残儿童和失智老年人常见病基础护理、社会福利机构经营与管理。

实习实训：对接真实职业场景或工作情境，在校内外进行生活照料、基础护理、康复训练等实训。在社会福利院、儿童福利院等基层社会福利机构进行岗位实习。

# 职业类证书举例

职业技能等级证书：老年照护、失智老年人照护、老年康体指导
