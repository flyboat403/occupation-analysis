# 7902 公共管理类

专业代码 790201

专业名称 民政服务

基本修业年限 三年

# 职业面向

面向社区工作者等职业，基层群众自治组织的民政协管（社会事务协管）等岗位（群）。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和民政政策法规与标准、民政与社会事务管理、社区社会福利服务、社区社会救助服务等知识，具备开展社会救助、优抚双拥、残疾人、老年人福利服务、慈善募捐服务、婚姻服务、殡葬服务等社区民政工作和社区社会事务管理能力，具有工匠精神和信息素养，能够从事基层民政服务等工作的技术技能人才。

# 主要专业能力要求

1. 具有基本的社会调查实践能力；  
2. 具有基本的民政公文写作能力；  
3. 具有民政政策法规与标准知识和民政执法能力；  
4. 具有民政服务、社会事务管理方面的知识和从事基层民政服务、社会事务管理的能力；  
5. 具有社区社会福利服务、社区社会救助服务的基本实践能力；  
6. 具有开展社会救助、优抚双拥、残疾人和老年人福利服务、慈善募捐服务、婚姻服务、殡葬服务等社区民政工作和社区社会事务管理的能力；  
7. 具有民政服务与社会事务管理方面的信息技术基础知识，具备运用民政服务领域信息化、数字化的能力；  
8. 具有团队协作精神和自主探索意识；  
9. 具有终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：行政管理、民政政策法规与标准、社会调查方法与统计、应用文写作。

专业核心课程：民政概论、社会福利实务、社区服务与管理实务、基层民政工作、社会救助工作、基层政权与社区建设、社区活动与策划、民政文化传播与新媒体。

实习实训：对接真实职业场景或工作情境，在校内外进行社会调查、社区民政业务、社区社会事务、社区社会福利和社会救助服务、社区社会工作等实训。在社区民政协管（社会事务协管）站、城乡社区村（居）委会、社区服务中心、社区社会福利机构等单位（场所）进行岗位实习。

# 职业类证书举例

职业技能等级证书：社区治理

# 接续专业举例

接续高职专科专业举例：民政服务与管理、婚庆服务与管理、智慧健康养老服务与管理

接续高职本科专业举例：民政管理、社会工作、行政管理

接续普通本科专业举例：社会工作、管理科学、行政管理

专业名称 产品质量监督检验

基本修业年限 三年

职业面向

面向检验员等职业，检验检测、过程品控、质量管理等岗位（群）。

培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和食品、农产品等理化检测、微生物检测与分析等知识，具备化工产品质量检验、生产安全管理与生产现场管理等能力，具有工匠精神和信息素养，能够从事检验检测、品质控制、质量管理等质量检验监督工作的技术技能人才。

# 主要专业能力要求

1. 具有利用检验检测实验室安全操作规范和质量管理规范的知识，进行标准操作规范（SOP）和保障检验设备设施安全的能力；  
2. 具有对检验原始数据进行处理，对检验结果做出分析，并在主检人员的指导下正确填写检验报告单的能力；  
3. 具有化学分析、微生物检验、光谱与色谱检验、仪器分析、快速检验等能力；  
4. 具有对食品、农产品等进行理化检测、微生物检测与分析，对食品原辅材料、半成品等进行检验检测的能力；  
5. 具有检测化工产品理化指标、成分等，使用、维护、保养主要检测设备和发现、判断、排除常用仪器设备常见故障的能力；  
6. 具有产品生产管理和品质控制、发现影响产品质量的异常情况、及时反馈并采取相应措施的能力；

7. 具有产品质量监督检验信息化管理和检验检测相关信息化操作的能力；  
8. 具有终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：实验室组织与管理、分析样品预处理及分类、基础化学、分析化学、微生物基础、质量管理。

专业核心课程：食品微生物检测技术、仪器分析技术、物理常数测定、农产品质量检测技术、化工产品质量检验、生产安全与现场管理。

实习实训：对接真实职业场景或工作情境，在校内外进行产品的取样、预处理、检验检测、质量分析以及过程质量管理等实训。在农产品、食品、化工产品生产企业及质量检验检测机构等单位（场所）进行岗位实习。
