# 7807 安全防范类

专业代码 780701

专业名称 安全保卫服务

基本修业年限 三年

# 职业面向

面向安全保卫服务人员等职业，人力防范服务、安全技术防范服务等岗位（群）。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和安全保卫、安防服务等知识，具备保安勤务执行、紧急情况处置、安防设施设备操作等能力，具有强烈的社会责任感、法治意识、忠诚奉献担当精神和信息素养，能够从事人力防范服务、安全技术防范服务等工作的技术技能人才。

# 主要专业能力要求

1. 具有熟练操作安防设施设备和使用安保器械的能力；  
2. 具有保安勤务执行和安全隐患识别的能力；  
3. 具有应对安保服务区域紧急情况处置和秩序维护的能力；  
4. 具有良好的沟通和协作的能力；  
5. 具有运用现代数字技术和手段提升安保服务的能力；  
6. 具有必备的体能和心理调试的能力；  
7. 具有终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：习近平法治思想概论、保安服务概述、保安法律基础、保安心理健康常识、消防安全基础。

专业核心课程：守护实务、巡逻实务、押运实务、安检实务、随身护卫实务、安全防范技术、紧急救助与现场保护、防卫技能训练。

实习实训：对接真实职业场景或工作情境，在校内外进行体能与队列训练、防卫技能训练、紧急救助与现场保护、守护、巡逻、押运、安检、随身护卫、安全防范技术等实训。在安保服务企业、企事业单位安保部门等单位或场所进行岗位实习。

# 职业类证书举例

职业资格证书：保安员、消防设施操作员

职业技能等级证书：安全防范系统建设与运维、智慧安防系统实施与运维、民航安全检查

# 接续专业举例

接续高职专科专业举例：安全保卫管理、智能安防运营管理、安全防范技术

接续高职本科专业举例：国际安保服务与管理、数字安防技术

接续普通本科专业举例：海外安全管理、国内安全保卫、安全防范工程
