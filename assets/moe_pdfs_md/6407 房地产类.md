# 6407 房地产类

专业代码 640701

专业名称 房地产营销

基本修业年限 三年

# 职业面向

面向房地产经纪人等职业，房地产销售代表、置业顾问、房地产抵押贷款专员、房地产权证办理专员、物业招商员、房地产网络经纪人等岗位（群）。

# 培养目标定位

本专业培养德智体美劳全面发展，掌握扎实的科学文化基础和房地产营销法律法规、房屋建筑、房地产经纪服务等知识，具备房地产营销市场调研、客户拓展与关系维护、房地产销售代理、居间和房地产交易代办等能力，具有工匠精神和信息素养，能够从事房地产经纪、房地产销售、商业物业招商、房地产抵押贷款服务、房地产权证办理服务及房地产网络顾问服务等工作的技术技能人才。

# 主要专业能力要求

1. 具有识读房屋建筑施工图和编制房屋状况说明书的能力；  
2. 具有进行房地产市场调查、分析调查数据和整理调查资料的能力；  
3. 具有完成客户拓展与关系维护、房地产买卖和租赁居间业务的能力；  
4. 具有从事房地产销售代理、二手物业租售、房地产网络销售、商业物业招商等业务的能力；  
5. 具有办理房地产税费业务、房地产抵押贷款及权证业务的能力；  
6. 具有适应产业数字化发展需求的基本数字技能和办公系统操作、房地产大数据平台运用、房地产在线数字营销的能力；  
7. 具有运用法律法规及相关政策解决房地产营销工作中实际问题的能力；  
8. 具有良好的语言表达能力、文字表达能力、沟通合作能力，较强的集体意识和团队合作意识；  
9. 具有终身学习和可持续发展的能力。

# 主要专业课程与实习实训

专业基础课程：房地产基础知识、建筑识图与构造、商务礼仪与沟通、办公软件应用。

专业核心课程：房地产市场调查、新建商品房经纪实务、存量房经纪实务、房地产 新媒体营销、商业物业招商实务、房地产抵押贷款实务、房地产法规应用。

实习实训：对接真实职业场景或工作情境，在校内外进行房地产销售实务、房地产经纪实务、房地产网络销售实务、商业物业招商实务、房地产抵押贷款实务等实训。在房地产经纪公司、房地产开发公司、房地产网络服务平台企业等单位进行岗位实习。

# 职业类证书举例

职业资格证书：房地产经纪专业人员职业资格

职业技能等级证书：新居住数字化经纪服务、数字营销技术应用

# 接续专业举例

接续高职专科专业举例：房地产经营与管理、房地产智能检测与估价、现代物业管理

接续高职本科专业举例：房地产投资与策划、现代物业管理

接续普通本科专业举例：房地产开发与管理、物业管理
